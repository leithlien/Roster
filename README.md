# Roster

## Intro
This project is inspired by my supervisor at my current workplace who spends 
excessive amounts of time creating rosters for employees. The system he and the
company uses offers no easy way to create a roster automatically, given a set of
employees and their availabilities. This problem is a classic example that can
be solved using network flow.
